# Number-Guessing-Game
A Python-based number guessing game where players attempt to guess a randomly generated number within a limited number of attempts. The game includes multiple difficulty levels, helpful hints, and replay functionality to enhance the user experience.

## Features
- Random number generation
- Difficulty levels (Easy, Medium, Hard)
- Limited attempts
- Hint system (higher/lower)
- Replay option

## Difficulty Levels

Easy
Range: 1–50  
Attempts: 10

Medium
Range: 1–100  
Attempts: 7

Hard
Range: 1–200  
Attempts: 5
